# Oh My Pi 0.10.5

This release establishes the first Oh My Pi Desktop package line.

## Highlights

- Oh My Pi branding, application identity, icons, and platform installers.
- Local-first OMP RPC sessions with dynamic model discovery.
- Session, tool, model, permission, and resource controls in a visual desktop shell.
- Windows, macOS, and Linux release packages built and verified in GitHub Actions.
- Windows installer smoke coverage for clean install, offline install, and upgrade paths.

## Installers

Download the package for your platform from the GitHub Release at https://github.com/BRCOO/ohmypi-craft/releases/tag/v0.10.5 and verify it with SHA256SUMS.txt.
